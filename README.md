# 👻 PACMAN 3D

## 📖 Sobre o Jogo
**PACMAN 3D** é uma recriação tridimensional do clássico jogo de arcade. O objetivo principal do jogador é navegar pelo labirinto, coletar todas as pastilhas espalhadas pelo mapa e sobreviver à perseguição dos fantasmas. Comer as super-pastilhas permite virar o jogo e capturar os fantasmas temporariamente!


## 🛠️ Tecnologias Utilizadas
* **Motor Gráfico:** Unity
* **Linguagem de Programação:** C#
